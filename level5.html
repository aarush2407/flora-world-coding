What is User Input?

Up until now, you have been the one writing all the data into the code. But real-world apps like Instagram or Netflix need to listen to the user. In Python, we use the input() function to pause the program and wait for the user to type something. This is the "Ear" of your program, allowing your code to gather information, names, or choices from the person sitting at the keyboard.

The input() function always captures information as a String. When you write name = input("What is your name? "), Python displays the question, catches the user's answer, and stores it inside the variable name. A key technical tip for Flora World Coding students: if you want the user to type a number for math, you must wrap your input in an int() function to convert it from text into a number!

Mastering input is the first step toward building interactive games and smart tools. Without input, software is just a static display. With it, you can create "Choose Your Own Adventure" stories, login screens, or personalized calculators. This interactivity is what makes programming so powerful‚Äîit allows you to build experiences that are different for every single person who uses them.
