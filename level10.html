Iterating with Intent

In Level 9, we used "While" loops to repeat code based on a condition. But what if you have a specific list of items or a set number of times you want to run your code? In Level 10, we introduce the For Loop. At Flora World Coding, we call this "definite iteration" because you know exactly when the loop will finish. It is the most common loop used by professional Python developers worldwide.

A for loop in Python is designed to go through a "sequence." Using the range() function, you can tell Python exactly how many times to sprout a new line of code. For example, for i in range(5): will run your code exactly five times. The variable i acts as a placeholder that keeps track of which "lap" the loop is currently on. This makes it incredibly efficient for handling data without the risk of an accidental infinite loop.

Why is this a favorite for AdSense-approved educational content? Because for loops are the engine behind data science and automation. From calculating the average temperature of a forest over a month to displaying every photo in a user's feed, for loops handle the heavy lifting. By mastering this, you are learning how to process large amounts of information with just two or three lines of code.

Key Concept: The range() Function

The range(start, stop) function generates numbers. Note that Python starts counting at 0 and stops one number before the stop value. So, range(5) gives you: 0, 1, 2, 3, 4.
