What is a While Loop?

In the previous levels, our code ran once and then stopped. But imagine if you had to print "I love Python" 1,000 times‚Äîyou wouldn't want to type it out manually! In Level 9, we introduce the While Loop. At Flora World Coding, we use loops to keep a program running as long as a certain condition is true. It‚Äôs like telling a gardener: "Keep watering the plants while the sun is still up."

[Image of Python while loop flowchart]

A while loop works very similarly to an if statement, but with a twist: it repeats. The computer checks the condition; if it's true, it runs the code inside and then jumps back to the top to check again. To prevent your program from running forever, you must make sure something inside the loop changes. This is usually done with a "counter" variable that increases or decreases until the condition finally becomes false.

Why are loops essential for AdSense-level apps? Think about a music player that repeats a song, or a game that keeps the "Game Over" screen hidden while the player has lives left. These are all loops. Mastering the while loop is the first step toward building automation scripts and complex game engines that need to constantly update information without crashing the computer.

If your condition is always true (like while 1 == 1:), your computer will get stuck in an "Infinite Loop." Always ensure your loop has a way to stop!
