What is a List?

Until now, every variable we created was like a single flower pot holding one flower. But in professional programming, we often need to manage hundreds of items at once. In Level 11, we introduce **Lists**. A list is a single variable that can hold multiple items, keeping them organized in a specific order. At **Flora World Coding**, we use lists to manage everything from a player's inventory to a database of plant species.

In Python, lists are created using square brackets `[ ]`, with items separated by commas. A critical technical concept to remember is **Indexing**. Python is a "zero-indexed" language, meaning the first item in your list is at position `0`, the second is at `1`, and so on. If you want to pull a specific item out of the list, you simply call the list name followed by its index number in brackets, like `my_list[0]`.

Lists are the backbone of data management. Think about your contact list on your phone or a shopping cart on an e-commerce site; these are all built using list-like structures. By mastering lists, you gain the ability to sort, filter, and modify large groups of data efficiently. This skill is highly valued in fields like data science and web development, making it a cornerstone of our Python curriculum.
